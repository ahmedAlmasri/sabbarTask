## Implementation Details

### Tech stack
* `Bloc` for state managment
* `Google map`
* Basic `Dependency injection` using `injectable`
* `fvm` flutter version `3.3.9`

## Running the Example
1. Clone this repository
2. Open `BleacherReportTask.xcodeproj` in Xcode.
